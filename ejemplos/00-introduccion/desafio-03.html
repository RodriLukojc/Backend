EJERCICIO 3

Crear un objeto en forma literal que cumpla con los siguientes puntos

  1. Debe contener una propiedad llamada 'meses' que almacene un array con los doce meses del año
  2. Implementará un método 'mostrarMeses' que muestre por consola los valores de dicho array mes a mes
  3. También se declarará un método 'getNumeroMes' que se le pasará el nombre del mes y retornará su número asociado (1 a 12)
  4. Por último, se declarará un método 'getLetrasMes' que devolverá un string con las primeras letras de cada mes
Llamar a las funciones con los parámetros correspondientes y mostrar sus respultados por consola

Aclaración: utilizar para la resolución, en lo posible, expresiones sencillas del lenguaje Javascript
